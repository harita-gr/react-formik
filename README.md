### Steps to Run:
- Clone the repository
  #### `git clone repo_url`
- Install all npm dependencies
  #### `npm i` 
- Run the app 
  #### `npm start` 
- The app starts running in <http://localhost:3000>
- Go to ``App.js``. Uncomment the code accordingly for each feature.
